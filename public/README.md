# Pasta Public - Assets Estáticos

## Logo

Para adicionar a logo da Estimulus:

1. Coloque o arquivo da logo na pasta `public/` com o nome `logo.svg` (ou `logo.png`, `logo.jpg`)
2. O componente Logo irá detectar automaticamente e usar a imagem
3. Se a logo não for encontrada, será exibida uma versão em texto estilizado

### Formatos suportados:
- SVG (recomendado para melhor qualidade)
- PNG
- JPG/JPEG

### Tamanho recomendado:
- Largura: 150px - 200px
- Altura: 40px - 60px
- Formato: Horizontal (landscape)

Você pode adicionar outras imagens, ícones e arquivos estáticos nesta pasta.
