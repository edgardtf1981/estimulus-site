# Logos de Clientes

Esta pasta contém os logos das empresas clientes da Estimulus.

## Como adicionar logos:

1. Coloque os arquivos de logo nesta pasta (`public/logos-clientes/`)
2. Formatos suportados: PNG, JPG, SVG
3. Nomeie os arquivos conforme os nomes definidos em `components/HomePage.tsx`:
   - `gambaru.png`
   - `papo-de-gestao.png`
   - `vendas.png`
   - `transformando-ideias.png`
   - `edgard-ferreira.png`
   - `cliente-6.png` (ou adicione mais conforme necessário)

## Tamanho recomendado:
- Largura: 150px - 200px
- Altura: 40px - 60px
- Formato: Horizontal (landscape) ou quadrado
- Fundo: Transparente (PNG) ou branco

## Estrutura atual:
Os logos serão carregados automaticamente. Se um logo não for encontrado, será exibido o nome da empresa como fallback.
